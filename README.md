# TRUNK: A Skin-Muscle-Inspired Shared Compliant Interface for Cooperative Multi-Mechanism Grasping

This repository contains the supplementary materials associated with the paper:

"TRUNK: A Skin-Muscle-Inspired Shared Compliant Interface for Cooperative Multi-Mechanism Grasping"

## Notice

The demonstration videos are currently being uploaded and may not yet be available in the repository. Additional grasping demonstrations and experimental videos will be added in a future update.

---

## Repository Structure

### Experiment-1

Contains the raw data used in the gripping interface evaluation experiment.

Contents:

- Statistical results used in Experiment 1

---

### Experiment-2

Contains the raw data used in the suction interface evaluation experiment.

Contents:

- Statistical results used in Experiment 2

---

### Experiment-3

Contains the rheological characterization data of the tested support media.

Contents:

- Flow curve measurements
- Amplitude sweep measurements
- 3ITT (Three-Interval Thixotropy Test) measurements
- Additional material characterization data

Note: The dataset also contains measurements labeled T400+SiO₂. This material is a modified formulation similar to T400. Because its rheological behavior was highly similar to that of T400, it was not included in the final analysis presented in the paper. The data are nevertheless provided here for completeness.

---

### Experiment-4

Contains the materials used in the system-level grasping evaluation.

Contents:

- Benchmark object information
- Object category references
- Representative grasping examples
- Experimental records used in Experiment 4
- Demonstration videos (uploading)

Note: Photographs of representative benchmark objects are provided in this folder. The complete object list, category labels, and experimental records are available in the accompanying data files.

---

### Experiment-5

Contains supplementary materials related to the ablation study and interface robustness evaluation.

Contents:

- PCB photograph
- System structure diagram

---

### TRUNK-Gripper

Contains supplementary materials related to the TRUNK robotic gripper.

Contents:

- System photographs
- Mechanical design files
- Additional implementation details

In addition, we have publicly released the power-distribution board currently used in the TRUNK platform. The PCB shown in the Experiment 5 hardware demonstration corresponds to this board.

For detailed PCB schematics and circuit design files, please refer to:

https://github.com/MagnumZhou/Trunk-Gripper_Powerboard_PCB
